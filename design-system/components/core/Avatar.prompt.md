Round initials chip with a stable per-user color — sidebar footer, table user cells, comment authors.

```jsx
<Avatar name="Dana Okoye" />
<Avatar name="admin" size={32} />
<Avatar initials="JS" color="#7a60e0" />
```

Derives initials and a palette color from `name`; override either with `initials` / `color`. `size` sets the pixel diameter (default 28).
